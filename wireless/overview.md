# Overview

## Encryption Types

* WEP (old)
* WPA (old)
* WPA2 PSK (personal)
  * Password-based
* WPA2 Enterprise (Mostly large companies)
  * Uses RADIUS or credentials
* WPA3 PSK (personal)
  * Password-based
* WPA3 Enterprise (Mosly large companies)
  * Uses RADIUS or credentials

## Process

### Place

Place wireless card into monitor mode

### Discover

Discover information about the network

* Channel
* BSSID

### Select

Select network and capture data

### Perform

Perform Deauth attack

### Capture

Capture WPA Handshake

### Attempt

Attempt to crack the handshake
